# WINQER Dashboard

Integrated Marketing Dashboard for WINQER clients.

## Tech Stack
- **Framework:** Next.js 13.5 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Shadcn UI
- **Backend:** Supabase (Planned)
- **AI:** OpenAI API (Planned)

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure
- `/src/app`: Pages and Layouts
- `/src/components`: UI Components
- `/src/lib`: Utilities
